# Qoder Orchestrator

Include this directory in your project or install globally.

## Files

- `qoder_orchestrator/` - Main package
- `subagents/` - Subagent definitions  
- `docs/` - Documentation
- `.qoder-orchestrate.yaml` - Example configuration
- `setup.py` - Installation script
- `install.sh` - Quick install script
- `requirements.txt` - Python dependencies

## Installation

See [INSTALL.md](INSTALL.md) for detailed instructions.

Quick install:
```bash
./install.sh
```

Or manually:
```bash
pip install -e .
```
